Cover Letter Generator

A web application that automatically generates personalized cover letters based on your CV and job descriptions, and sends them via Gmail using local Ollama AI.

Python Ollama Gmail API PDF Processing Flask

A web app that auto-generates personalized cover letters from your CV and job descriptions, then sends them via Gmail.

  • Upload CV (PDF) for automatic parsing
  • Input job description and desired word count
  • AI-powered cover letter generation using local Ollama (llama2)
  • Email preview with AI subject line and Gmail API sending

Built with Python, Flask, Ollama, and Gmail API. All processing is local—ensuring privacy and security.

Library Management System

A full-stack web application for managing library operations including book inventory, user management, and borrowing/returning functionality.

Vue.js 3 Express.js MongoDB JWT Vite

A comprehensive library management system with full CRUD operations and role-based access control.

  • Book management: CRUD for books (add, edit, delete, view details), search, and browse
  • User management: Admin-only user CRUD, role-based access (admin/regular), and authentication
  • Borrow/return: Borrow and return books, track borrow history per user and book, and manage availability
  • Authentication & authorization: JWT-based login/logout, protected routes, and admin-only features

Built with Vue.js 3, Express.js, MongoDB, JWT authentication, and Vite for a modern full-stack experience.
